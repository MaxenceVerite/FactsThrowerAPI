# Facts Throwing API
> Version française
## But
Facts Throwing API est une API Rest que j'ai réalisé sur mon temps libre. Comme son nom l'indique, elle a été réalisé dans le but de mettre à disposition des faits de connaissances générales 
Seul des faits en langue française sont fournis pour le moment. Il est possible qu'à l'avenir soit proposée une traduction anglaise.

### Tech

* | [.NET CORE] | - Framework C# 
* | [Entity Framework Core] | - Un très bon framework d'accès aux données (couplé avec Fluent API pour la description des relations de la base de données)
* | [Microsoft SQL Server] | - Service de base de données de Microsoft 
* | [Swagger] | - Outil super pratique pour la documentation de l'API, le test des requêtes et la lisibilité pour l'utilisateur
